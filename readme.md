Lemon Party (NSFW)
==================

A game of fantasy football gone horribly wrong.

## You Need

- [ruby](https://www.ruby-lang.org/en/)
- [bundler](http://bundler.io/)

## Setup

- `bundle install`
- install [pow](http://pow.cx/): `rake pow:install`
- `rake pow:link`
- `cp .env.example .env` and update with your ENV variables
- `rake open`

## Tests

- `rake`
